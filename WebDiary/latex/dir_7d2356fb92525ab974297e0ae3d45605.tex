{\ttfamily \texorpdfstring{$<$}{<}major\texorpdfstring{$>$}{>}.\texorpdfstring{$<$}{<}minor\texorpdfstring{$>$}{>}.\texorpdfstring{$<$}{<}patch\texorpdfstring{$>$}{>}}

And constructed with the following guidelines\+:


\begin{DoxyItemize}
\item Breaking backward compatibility bumps the major (and resets the minor and patch)
\item New additions, including new icons, without breaking backward compatibility bumps the minor (and resets the patch)
\item Bug fixes, changes to brand logos, and misc changes bumps the patch
\end{DoxyItemize}
